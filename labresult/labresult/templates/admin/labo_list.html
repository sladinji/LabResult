Paramétrer les laboratoires

Quels sont les paramétrages possibles pour un laboratoire ?

Pour chaque laboratoire, il est possible d'associer des imprimantes pour chaque type de document. Les impressions peuvent être lancées de manière automatique ou manuelle. Il est également possible d'ajouter une entête pour chaque type de document afin d'avoir exactement le même rendu entre la version imprimée et celle numérique.
